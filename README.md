# Decision Tree Explorer

Interactive career roadmap visualization -- 7 life paths for an Indian commerce student (age 18 to 40).

**Live demo:** https://ayyushnegii.github.io/Decision-tree/

## Features

- **159 nodes** across 4 streams (Science, Commerce, Arts, Universal)
- **108 end outcomes** with salary, net worth, probability, and risk data
- **4 visualization views:**
  - **Tree View** -- horizontal level-based node layout
  - **Path Explorer** -- click through the tree, see outcome details
  -- **SVG Tree** -- full scalable vector graphic rendering
- ** Stats view** -- stream breakdown, top outcomes, risk distribution
- Dark neon cyberpunk UI

## Files

- `index.html` -- main page with embedded CSS
- `tree-data.js` -- all tree nodes, choices, and outcome data
- `app.js` -- rendering logic for all 4 views

## Data Source

All data extracted from [ayyushnegii/7-path-decision-tree](https://github.com/ayyushnegii/7-path-decision-tree). No original data was modified.

## How to Run

Just open `index.html` in a browser. No server needed.
